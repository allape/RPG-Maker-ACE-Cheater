# Build

This script should be run in Windows.

```shell
gem install scanf ocran
gem install psych -v 2.0.0

ocran rvunpacker.rb
```

# Usage

```shell
ruby rvunpacker.rb [decode|encode] [game directory]
rvunpacker.exe [decode|encode] [game directory]
```
